# RMRP Auto Hot Key 1.0 Beta
Авто Хот Кей созданный в роли памятки для государственных сотрудников на проекте RMRP.

Version: 1.0 Beta (Програ не имеет возможности обновляться автоматически, по этому следите за обновлениями на гитхабе)

# FAQ
Вопрос: К кому обратиться в случае если я найду ошибку или захочу предложить идею?
Ответ: Можно обратиться в дискорд канал создателя https://discord.gg/KknQRacwHE

Вопрос: Есть ли возможность получить исходный код для улучшения?
Ответ: Да можно. Но это обговаривается с создателем в дискорде (@ckolerbb)

Вопрос: Можно ли использовать АХК на других проектах?
Ответ: Можно, но для этого нужно связаться с создателем, для того что бы он отправил вам ссылку на дополнение к ахк.

Вопрос: Можно ли помочь в разработке АХК?
Ответ: Конечно, принять участвовать в разработке можно в дискорд канале создателя https://discord.gp/KknQRacwHE

Вопрос: Меня не забанят за АХК?
Ответ: Нет, это не считается за чит или какое либо ПО дающее преимущество, просто памятка с встроенным АХК для отправки жетона/представления.

Вопрос: Если я получу исходный код, могу ли я его продать?
Ответ: Да можете.
