# FAQ
Вопрос: К кому обратиться в случае если я найду ошибку или захочу предложить идею?
Ответ: Можно обратиться в дискорд канал создателя https://discord.gg/KknQRacwHE

Вопрос: Можно ли использовать АХК на других проектах?
Ответ: Можно, ахк имеет открытый код, который можно редактировать

Вопрос: Меня не забанят за АХК?
Ответ: Нет, это не считается за чит или какое либо ПО дающее преимущество, просто памятка с встроенным АХК для отправки жетона/представления.
